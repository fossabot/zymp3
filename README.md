
##Zymp3  
####A frontend for youtube-dl and ffmpeg/avconv



###Dependencies  


* youtube-dl
* zenity or yad
* xdg-utils
* libnotify
* ffmpeg or avconvert


###Install  


A install file is included, just run:  
  
```./install.sh```  

  
###Uninstall  

To uninstall run:  

```./uninstall.sh```
